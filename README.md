# 📝 DevInsights Blog Website

## 📌 Project Description
DevInsights Blog is a **responsive and interactive blog website** built using **HTML, CSS, and JavaScript**.

It allows users to:
- Read blog posts
- Search content dynamically
- Add comments to individual posts

This project demonstrates **DOM manipulation, event handling, and UI design** without using any external frameworks.

---

## 🚀 Features
- ✅ Multiple blog posts with navigation
- ✅ Sticky header with navigation menu
- ✅ Live search functionality (filters posts instantly)
- ✅ Comment system for each blog post
- ✅ Dynamic DOM updates (no page reload)
- ✅ Clean and responsive UI design
- ✅ Organized layout using modern CSS

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout (Flexbox, variables, responsive design)  
- **JavaScript (Vanilla JS)** – Interactivity and dynamic content

## 🎮 How It Works

### 🔍 Search Feature
- As the user types in the search bar  
- Blog posts are filtered in real-time  
- Only matching posts are displayed  

### 💬 Comment System
- Each blog post has its own comment section  
- Users can enter:
  - Name  
  - Comment  
- Comments are displayed instantly without reloading  

---

## 🧠 Core Logic

### Comment Handling
- Uses `addEventListener()` on form submission  
- Creates new comment elements dynamically  
- Appends comments to the respective post  

### Search Functionality
- Captures user input from search bar  
- Converts text to lowercase for matching  
- Shows/hides posts based on content match
